# 💸 Expense Tracker

A simple and clean web-based expense tracker that helps you log, split, and manage your daily expenses — built with pure HTML, CSS, and JavaScript.

---

## 🚀 Features

- Add expenses with a name and place
- Choose between **Full** or **Split** contribution
- Automatically calculates each person's share for split expenses
- Live **running total** in Indian Rupees (Rs.)
- **Delete** individual expenses and auto-update the total
- **Enter key** support for faster input
- Fully responsive layout for mobile and desktop

## ▶️ How to Run

1. **Clone or download** this repository
2. Open `index.html` in any modern web browser
3. No installation or dependencies required!

```bash
git clone https://github.com/your-username/expense-tracker.git
cd expense-tracker
open index.html
```

---

## 📖 How to Use

1. Enter the **Expense Name** (e.g., Dinner)
2. Enter the **Place** (e.g., Pizza Hut)
3. Enter the **Amount** in Rs.
4. Select **Full** (you paid everything) or **Split** (shared among people)
5. If Split, enter the **Number of People**
6. Click **Calculate** or press **Enter**
7. The expense appears in the list with a running total
8. Click **Delete** on any item to remove it and update the total
